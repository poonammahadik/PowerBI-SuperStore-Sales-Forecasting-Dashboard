# Super Store Sales Forecasting Dashboard

An interactive Power BI dashboard built to analyze sales 
performance and forecast future sales trends for a Super Store
across multiple regions, segments, and product categories.

## Dashboard Pages
- **Page 1 - Dashboard:** Complete sales performance overview
- **Page 2 - Forecasting:** 15-day sales forecast using time series

## Dashboard Highlights
- **Total Orders:** 5.90K
- **Total Sales:** 1.57M
- **Total Profit:** 0.18M
- **Average Order Days:** 4

## Features
- KPI cards for Orders, Sales, Profit and Order Days
- Sales breakdown by Segment (Consumer, Corporate, Home Office)
- Sales breakdown by Payment Mode (COD, Online, Cards)
- Sales breakdown by Region (Central, East, South, West)
- Monthly Sales and Profit trends (2019 vs 2020)
- Sales by Sub-Category (Phones, Storage, Tables, etc.)
- Sales by Ship Mode and Category
- State-wise sales map across North America
- 15-day sales forecasting with confidence interval
- Sum of sales by state (Texas, Washington, Virginia, etc.)
- Interactive region filters for dynamic exploration

## Tools Used
Power BI Desktop

## Skills Demonstrated
Data Visualization, Time Series Forecasting, DAX Measures,
KPI Design, Maps, Slicers and Filters, Business Insights
